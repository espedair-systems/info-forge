# Forge

## Installation

### Windows
1. Download the `Schema Forge Setup.exe` installer from the releases.
2. Double-click the `.exe` file to run the installer.
3. Follow the setup wizard to select your installation path.
4. An application shortcut will be created on your Desktop and in the Start Menu.

### Linux
1. Download the Debian package file (`.deb`).
2. Install it using your package manager or command line:
   ```bash
   sudo dpkg -i schema-forge_*_amd64.deb
   sudo apt-get install -f  # resolves any missing dependencies
   ```
3. Launch the application from your desktop environment's application launcher.

### macOS
1. Download the ZIP file for your system architecture:
   - **Intel Macs**: `Schema Forge-mac-x64.zip`
   - **Apple Silicon (M1/M2/M3)**: `Schema Forge-mac-arm64.zip`
2. Extract the downloaded ZIP file to get `Schema Forge.app`.
3. Drag the app into your `Applications` folder.
4. Double-click to launch it (you may need to approve the developer signature in System Settings under Privacy & Security).
